# Autonomous AI Form Filler

This Python script automates the process of filling out a simple form using Selenium for web automation and OpenAI's GPT-3.5 model for generating content. It fills out the form fields with dummy data and generates interesting text for specific prompts.

## Instructions

1. Clone this repository to your local machine:

    ```bash
    git clone <https://github.com/NiranjanRathor47/Autonomous-AI-Form-Filler/>
    ```

2. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Set up your OpenAI API key:

    - Replace the placeholder `'Your_API_Key'` in the script with your actual OpenAI API key.

4. Run the Python script:

    ```bash
    python form_filler.py
    ```

5. Follow the prompts to fill in the form with generated data and submit it.

## Important Notes

- Make sure to replace the placeholder API key with your actual OpenAI API key. Do not share or publish your API key on GitHub.
- The script uses Selenium for web automation and OpenAI's GPT-3.5 model for text generation.
- Ensure you have Chrome WebDriver installed and its path configured properly.
- The generated content might vary based on the prompt and the AI model's responses.
